# CodeClicker(コードクリッカー)へようこそ！
## 概要
クリックでコーディングをしてお金を稼ぎましょう！<br>
稼いだお金でコーディングを自動化してさらに効率的に稼ごう！

*このゲームはJavaScriptの学習のために作成したゲームです*<br>
*不具合や問題などがあるかもしれませんがそっと教えてください*

### 2026/6/25
- リファクタリング（プレステージ機能）を追加
    - 所持金額が一定以上に達すると現在の強化をリセットし、所持金額に応じたコミットログを得るリファクタリングを実装しました
    - コミットログを消費して購入できる特殊な強化を実装しました
    - コミットログは1つあたり自動獲得量に1%のボーナスを与え、消費することでリファクタリング時に失われない特別な強化を購入することができます
    - 現時点で実装した特殊強化は2つです
- 表示の桁を無量大数(10^68)まで対応しました
